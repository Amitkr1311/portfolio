# Portfolio Website

A modern, animated portfolio website built with Next.js, GSAP, and Tailwind CSS.

## Features

- 🎨 Beautiful gradient designs with Tailwind CSS
- ✨ Smooth GSAP animations and scroll-triggered effects
- 📱 Fully responsive design
- ⚡ Built with Next.js 14 and TypeScript
- 🎯 Smooth scrolling navigation
- 💼 Project showcase section
- 📧 Contact form
- 🌙 Dark theme optimized

## Tech Stack

- **Framework:** Next.js 14
- **Styling:** Tailwind CSS
- **Animations:** GSAP (GreenSock Animation Platform)
- **Language:** TypeScript
- **Icons:** Lucide React

## Getting Started

1. Install dependencies:
```bash
npm install
```

2. Run the development server:
```bash
npm run dev
```

3. Open [http://localhost:3000](http://localhost:3000) in your browser

## Sections

- **Hero:** Eye-catching introduction with animated text
- **About:** Personal information and statistics
- **Skills:** Technology stack with skill levels
- **Projects:** Featured project showcase
- **Contact:** Get in touch form

## Customization

Edit the content in the component files:
- `app/components/Hero.tsx` - Update name and social links
- `app/components/About.tsx` - Modify about text and stats
- `app/components/Skills.tsx` - Add/edit your skills
- `app/components/Projects.tsx` - Showcase your projects
- `app/components/Contact.tsx` - Update contact information

## Build for Production

```bash
npm run build
npm start
```
